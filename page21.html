unit 1, outcome 3:

3.4 Describe how you would go about testing digital solutions


I would test digital solutions by running the html files in the browser. I would run these files in a number of browsers to get a better perspective of how they look, to see if there’s continuity within the webpages, to make sure there’s no problems with any text or images or links used on each of the pages. Any alterations that would be made to the pages would need to be saved and then the adjusted html files would then have to be tested in the browsers all over again to make sure for regularities.
